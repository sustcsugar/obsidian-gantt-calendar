# 任务视图 - 状态过滤演示

本文档演示任务视图的状态过滤功能。通过切换不同的状态筛选器，可以查看不同状态的任务。

## 支持的任务状态

插件支持以下 7 种默认状态：

| 符号 | 状态 | 说明 |
|------|------|------|
| `[ ]` | 待办 (todo) | 未开始的普通任务 |
| `[x]` | 已完成 (done) | 已完成的任务 |
| `[!]` | 重要 (important) | 重要任务 |
| `[-]` | 已取消 (canceled) | 已取消的任务 |
| `[/]` | 进行中 (in_progress) | 正在进行的任务 |
| `[?]` | 有疑问 (question) | 有疑问需要确认的任务 |
| `[n]` | 已开始 (start) | 已开始但未标记为进行中 |

## 使用方法

### 在任务视图中筛选状态

1. 打开插件的**任务视图**
2. 在工具栏找到**状态筛选**下拉菜单
3. 选择要查看的状态：
   - **全部** - 显示所有任务
   - **待办** - 显示未开始的普通任务
   - **已完成** - 显示已完成的任务
   - **进行中** - 显示正在进行的任务
   - **重要** - 显示标记为重要的任务
   - **已取消** - 显示已取消的任务
   - **有疑问** - 显示有疑问的任务
   - **已开始** - 显示已开始的任务

### 状态切换

您可以通过修改复选框内容来切换任务状态：

```markdown
# 从待办改为进行中
- [ ] 任务标题    =>    [/] 任务标题

# 从进行中改为已完成
- [/] 任务标题    =>    [x] 任务标题

# 从待办改为取消
- [ ] 任务标题    =>    [-] 任务标题
```

## 快捷操作

### 批量操作

在任务视图中，您可以：
1. 选中多个任务
2. 批量修改状态
3. 批量删除或移动任务

### 快速筛选

使用状态筛选器可以：
- 快速查看特定状态的任务
- 组合其他筛选条件使用
- 保存常用的筛选组合
