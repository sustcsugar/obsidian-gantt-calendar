# 任务视图 - 搜索功能演示

本文档演示任务视图的搜索和过滤功能，帮助您快速找到需要的任务。

## 搜索功能概述

任务视图支持以下搜索方式：
1. **关键词搜索** - 在任务标题中搜索关键词
2. **标签搜索** - 按标签过滤任务
3. **路径搜索** - 按文件路径过滤任务
4. **组合搜索** - 多个条件组合使用

## 高级搜索技巧

### 模糊搜索
- [ ] 搜索 "dev" 可匹配 "development" "developer" "develop"
- [ ] 搜索 "test" 可匹配 "testing" "tester"

### 精确匹配
- [ ] 用引号包裹进行精确搜索："API开发"
- [ ] 只匹配完全相同的关键词

### 排除关键词
- [ ] 搜索 "开发 -测试" - 包含"开发"但不包含"测试"

### OR 搜索
- [ ] 搜索 "开发 OR 设计" - 匹配包含"开发"或"设计"的任务

## 搜索快捷操作

### 保存搜索条件

常用的搜索条件可以保存为快捷筛选器：

1. **我的工作** - 标签 #work + 待办状态
2. **紧急任务** - 优先级最高 + 高
3. **本周到期** - 到期日期在本周内
4. **学习任务** - 标签 #learning
5. **每日笔记任务** - 路径 dailynote/

### 搜索历史

最近使用的搜索条件会保存在历史记录中，方便快速重复使用。

## 搜索结果排序

搜索结果支持多种排序方式：

- **按相关性** - 最相关的结果在前
- **按到期日期** - 即将到期的在前
- **按优先级** - 高优先级的在前
- **按创建日期** - 最新创建的在前
