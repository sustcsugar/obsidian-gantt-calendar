# 甘特图 - 基础功能演示

甘特图用于可视化项目任务的时间安排和依赖关系，是项目管理的有力工具。

## 甘特图特点

- **时间轴视图** - 横向时间轴显示任务跨度
- **任务条** - 可视化显示任务持续时长
- **进度显示** - 显示任务完成百分比
- **依赖关系** - 显示任务间的前后关系
- **里程碑** - 标记重要节点
- **当前日期线** - 显示今天的位置

## 甘特图操作

### 视图导航

- **时间轴缩放** - 调整显示的时间范围（日/周/月）
- **左右滚动** - 查看不同时间段的任务
- **今天按钮** - 快速定位到当前日期
- **任务搜索** - 快速找到特定任务

### 任务操作

- **查看详情** - 点击任务条查看详细信息
- **编辑任务** - 修改任务时间和属性
- **拖拽调整** - 拖拽任务条调整时间
- **调整时长** - 拖拽任务条边缘调整持续时长

### 显示选项

- **显示任务名称** - 在任务条上显示任务名称
- **显示日期** - 显示任务的开始和结束日期
- **显示进度** - 显示任务完成进度
- **显示优先级** - 用颜色区分优先级
- **显示里程碑** - 高亮显示里程碑任务
- **网格线** - 显示时间网格线

## 甘特图最佳实践

### 任务规划

1. **设置合理时长** - 不要低估任务所需时间
2. **添加缓冲时间** - 在任务间预留缓冲
3. **明确里程碑** - 清晰标记重要节点
4. **避免过度并行** - 有限的资源下不要安排过多并行任务

### 进度管理

1. **定期更新** - 每天或每周更新任务状态
2. **跟踪延期** - 标记延期的任务并分析原因
3. **调整计划** - 根据实际情况调整后续任务
4. **沟通协调** - 及时与团队成员沟通进度

### 可视化技巧

1. **颜色编码** - 用颜色区分任务类型和状态
2. **优先级标识** - 高优先级任务更醒目
3. **里程碑突出** - 用特殊样式显示里程碑
4. **进度清晰** - 清晰显示任务完成百分比
