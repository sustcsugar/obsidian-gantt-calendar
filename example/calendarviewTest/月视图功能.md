# 日历视图 - 月视图功能

月视图提供单月的详细任务视图，是日常任务管理的主要视图。

## 月视图特点

- **日期网格** - 清晰显示本月每一天
- **任务预览** - 在日期格中显示任务摘要
- **周数显示** - 显示当前周数
- **今日高亮** - 高亮显示今天的日期
- **跨月显示** - 显示上月末和下月初的任务

## 视图操作

### 导航操作

- **上个月** - 点击工具栏的上月按钮
- **下个月** - 点击工具栏的下月按钮
- **回到今天** - 点击"今天"按钮快速定位到今天
- **周视图** - 双击某周进入该周的周视图
- **日视图** - 点击某天进入该天的日视图

### 任务操作

- **创建任务** - 点击日期格打开任务创建面板
- **查看任务** - 点击任务查看详情
- **编辑任务** - 在任务详情中编辑
- **拖拽任务** - 拖拽任务到新的日期更改到期日期

### 显示选项

- **显示周数** - 在视图中显示周数
- **显示周末** - 高亮显示周末
- **任务数量限制** - 每个日期格最多显示的任务数
- **紧凑模式** - 减小间距显示更多内容

## 日期信息显示

### 日期格内容

每个日期格中可以显示：
- **日期数字** - 当天的日期
- **任务数量** - 当天的任务总数
- **任务列表** - 任务摘要（标题前几个字）
- **状态标识** - 不同颜色表示不同状态
- **优先级标识** - 优先级 emoji

### 颜色编码

- 🔴 **红色边框** - 今天
- 🟠 **橙色标记** - 高优先级任务
- 🔵 **蓝色标记** - 中优先级任务
- ⚪ **灰色标记** - 低优先级任务
- 🟢 **绿色标记** - 已完成任务
- 🟡 **黄色标记** - 进行中任务

## 月视图技巧

### 高效规划

1. **月初规划** - 每月初花时间规划整个月的任务
2. **预留缓冲** - 不要把每天排满，预留应急时间
3. **里程碑标记** - 用特殊标记突出重要日期
4. **定期回顾** - 每周回顾月度计划，调整安排

### 可视化技巧

1. **颜色编码** - 用不同颜色区分任务类型
2. **优先级标识** - 高优先级任务更显眼
3. **任务分组** - 相关任务放在相邻日期
4. **平衡负载** - 避免某天任务过多

### 时间管理

1. **深度工作块** - 上午安排需要专注的任务
2. **会议时间** - 下午集中安排会议
3. **学习时间** - 晚上或周末安排学习
4. **休息时间** - 确保有足够的休息日

## 常见问题

**Q: 如何在月视图中创建全天事件？**
A: 创建一个没有具体时间的任务，它会显示在日期格的顶部。

**Q: 月视图最多能显示多少个任务？**
A: 每个日期格默认显示最多3个任务，多余的任务会以"+N"的形式显示。

**Q: 如何打印月视图？**
A: 使用浏览器的打印功能或 Obsidian 的导出功能。
