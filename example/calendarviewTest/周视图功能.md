# 日历视图 - 周视图功能

周视图提供单周的详细任务视图，适合周度任务规划和日常任务管理。

## 周视图特点

- **7天视图** - 清晰显示本周7天
- **时间轴** - 可选的时间轴视图
- **任务详情** - 显示更多任务信息
- **拖拽调度** - 方便地拖拽任务重新安排
- **周统计** - 显示本周任务完成情况

## 视图操作

### 导航操作

- **上一周** - 点击工具栏的上周按钮
- **下一周** - 点击工具栏的下周按钮
- **回到本周** - 点击"今天"按钮回到当前周
- **日视图** - 点击某天进入该天的日视图
- **月视图** - 点击工具栏切换到月视图

### 任务操作

- **创建任务** - 点击日期格或拖拽创建任务
- **查看详情** - 点击任务查看完整信息
- **编辑任务** - 在详情面板中编辑
- **拖拽移动** - 拖拽任务到新的日期
- **复制任务** - 复制任务到其他日期

### 显示选项

- **显示周末** - 高亮显示周六日
- **显示周数** - 显示当前周数
- **任务排序** - 按优先级或时间排序
- **时间轴显示** - 开启时间轴视图

## 周视图最佳实践

### 周一规划

1. **回顾上周** - 检查上周未完成的任务
2. **本周目标** - 设定本周要完成的 3 个关键目标
3. **任务分解** - 将大任务分解到具体日期
4. **预留缓冲** - 不要把每天排满

### 每日执行

1. **每日优先** - 每天确定最重要的 3 件事
2. **专注完成** - 一次只做一件事
3. **定期检查** - 每半天检查一次进度
4. **灵活调整** - 根据实际情况调整计划

### 周五总结

1. **完成回顾** - 回顾本周完成的任务
2. **未完成分析** - 分析未完成任务的原因
3. **经验总结** - 记录本周学到的经验
4. **下周预备** - 为下周做好准备
