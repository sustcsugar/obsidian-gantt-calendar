# 日历视图 - 日视图功能

日视图提供单日的详细任务视图，适合精确的时间规划和每日任务管理。

## 日视图特点

- **时间轴** - 完整的 24 小时时间轴
- **任务块** - 可视化显示任务时长
- **精确时间** - 设置任务的开始和结束时间
- **拖拽调整** - 拖拽调整任务时间和时长
- **今日统计** - 显示今日任务完成情况

## 视图操作

### 导航操作

- **前一天** - 点击工具栏的前一天按钮
- **后一天** - 点击工具栏的后一天按钮
- **回到今天** - 点击"今天"按钮
- **跳转到日期** - 点击日期选择器

### 任务操作

- **创建任务** - 点击时间轴空白处创建任务
- **拖拽移动** - 拖拽任务到新的时间
- **调整时长** - 拖拽任务边缘调整时长
- **查看详情** - 点击任务查看详情
- **快速编辑** - 双击任务快速编辑

### 显示选项

- **时间范围** - 设置显示的时间范围（如 6:00-24:00）
- **时间间隔** - 设置时间轴刻度（15/30/60分钟）
- **当前时间线** - 显示当前时间线
- **已完成任务** - 显示/隐藏已完成任务
- **任务颜色** - 按状态/优先级/标签着色

## 时间轴显示

### 时间刻度

日视图支持多种时间刻度：
- **15分钟** - 最精细，适合精确规划
- **30分钟** - 默认，适合大多数场景
- **1小时** - 最粗略，适合概览

### 当前时间线

红色虚线显示当前时间位置：
- 自动更新
- 快速定位现在
- 查看剩余任务

### 任务块显示

任务块显示信息：
- **任务标题** - 任务名称
- **时间范围** - 开始和结束时间
- **时长** - 任务持续时长
- **颜色** - 按状态/优先级
- **标记** - 重要标记、标签等

## 日视图最佳实践

### 每日规划

1. **前一天晚上** - 规划第二天的主要任务
2. **当天早上** - 检查计划，做微调
3. **设置优先级** - 确定 3 个最重要的任务
4. **预留缓冲** - 不要把时间排满

### 执行建议

1. **专注当前** - 一次只做一件事
2. **能量管理** - 在高能量时做重要任务
3. **定期休息** - 每 1-2 小时休息一次
4. **灵活调整** - 根据实际情况调整计划

### 每日回顾

1. **完成任务** - 标记已完成的任务
2. **未完成分析** - 记录未完成原因
3. **时间记录** - 回顾时间使用情况
4. **明日计划** - 为明天做好准备
