# 日历视图 - 年视图功能

年视图提供全年的任务概览，方便进行长期规划和回顾。

## 年视图特点

- **全年概览** - 一眼看到全年的任务分布
- **月份导航** - 快速跳转到任意月份
- **任务密度指示** - 通过颜色深浅显示每月任务数量
- **年份切换** - 快速切换不同年份
- **快速跳转** - 点击月份切换到月视图

## 视图操作

### 导航操作

- **上一年** - 点击工具栏的上一年按钮
- **下一年** - 点击工具栏的下一年按钮
- **回到今年** - 点击"今天"按钮回到当前年份
- **月份跳转** - 点击任意月份进入该月的月视图

### 显示选项

- **显示周数** - 在视图中显示周数
- **任务数量显示** - 在每个月格中显示任务数量
- **颜色编码** - 根据任务状态/优先级显示不同颜色

### 筛选选项

在年视图下可以进行以下筛选：
- **状态筛选** - 只显示特定状态的任务
- **优先级筛选** - 只显示特定优先级的任务
- **标签筛选** - 只显示特定标签的任务

## 提示和技巧

### 年视图最佳实践

1. **宏观规划** - 用年视图做长期规划，不要陷入细节
2. **里程碑标记** - 将重要里程碑设为高优先级，在年视图更醒目
3. **平衡负载** - 通过年视图检查任务分布是否均匀
4. **预留缓冲** - 避免在短时间内安排过多任务

### 颜色编码建议

- 🔴 **红色** - 关键里程碑、截止日期
- 🟠 **橙色** - 高优先级任务
- 🔵 **蓝色** - 常规任务
- 🟢 **绿色** - 已完成任务
- ⚪ **灰色** - 低优先级任务

### 常见问题

**Q: 年视图能显示具体任务内容吗？**
A: 年视图主要用于概览，显示任务数量和状态。要查看具体任务，请点击月份进入月视图。

**Q: 如何在年视图中创建跨月任务？**
A: 可以在任意文件中创建带有开始日期和到期日期的任务，任务会自动显示在跨越的月份中。

**Q: 年视图支持多年显示吗？**
A: 当前年视图只显示单年，但可以通过年份切换按钮查看不同年份。
