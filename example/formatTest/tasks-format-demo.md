# Tasks 格式 (Emoji) 完整演示

本文档演示了 Tasks 插件格式的所有功能，使用 Emoji 标记来表示优先级、日期等信息。

## 基本格式

### 最简单的任务
```markdown
- [ ] 这是一个待办任务
- [x] 这是一个已完成任务
```

示例：
- [ ] 这是一个待办任务
- [x] 这是一个已完成任务

## 优先级

### 优先级 Emoji 对照表

| Emoji | 优先级 | 说明 |
|-------|--------|------|
| 🔺 | 最高 (highest) | 最紧急重要的任务 |
| ⏫ | 高 (high) | 需要优先处理的任务 |
| 🔼 | 中 (medium) | 普通优先级任务 |
| 🔽 | 低 (low) | 可以延后处理的任务 |
| ⏬ | 最低 (lowest) | 有空再做的任务 |

### 优先级示例

- [ ] 最高优先级任务 🔺
- [ ] 高优先级任务 ⏫
- [ ] 中等优先级任务 🔼
- [ ] 低优先级任务 🔽
- [ ] 最低优先级任务 ⏬

## 日期类型

### 日期 Emoji 对照表

| Emoji | 日期类型 | 说明 |
|-------|----------|------|
| ➕ | 创建日期 (created) | 任务创建的时间 |
| 🛫 | 开始日期 (start) | 任务开始执行的时间 |
| ⏳ | 计划日期 (scheduled) | 计划执行的时间（也支持 ⌛） |
| 📅 | 到期日期 (due) | 任务截止时间（也支持 📆、🗓） |
| ✅ | 完成日期 (completion) | 任务完成的时间 |
| ❌ | 取消日期 (cancelled) | 任务被取消的时间 |

### 日期格式示例

- [ ] 带创建日期的任务 ➕ 2024-01-10
- [ ] 带开始日期的任务 🛫 2024-01-15
- [ ] 带计划日期的任务 ⏳ 2024-01-20
- [ ] 带到期日期的任务 📅 2024-01-25
- [x] 带完成日期的任务 ✅ 2024-01-10
- [-] 带取消日期的任务 ❌ 2024-01-15

## 任务状态

### 状态符号对照表

| 符号 | 状态 | 说明 |
|------|------|------|
| `[ ]` | 待办 (todo) | 未开始的普通任务 |
| `[x]` | 已完成 (done) | 已完成的任务 |
| `[!]` | 重要 (important) | 重要任务 |
| `[-]` | 已取消 (canceled) | 已取消的任务 |
| `[/]` | 进行中 (in_progress) | 正在进行的任务 |
| `[?]` | 有疑问 (question) | 有疑问需要确认的任务 |
| `[n]` | 已开始 (start) | 已开始但未标记为进行中 |

### 状态示例

- [ ] 待办任务
- [x] 已完成任务
- [!] 重要任务
- [-] 已取消任务
- [/] 进行中任务
- [?] 有疑问任务
- [n] 已开始任务

## 同义 Emoji 支持

### 到期日期同义词
- [ ] 使用 📅 的任务 ➕ 2024-01-10 📅 2024-01-15
- [ ] 使用 📆 的任务 ➕ 2024-01-10 📆 2024-01-15
- [ ] 使用 🗓 的任务 ➕ 2024-01-10 🗓 2024-01-15

### 计划日期同义词
- [ ] 使用 ⏳ 的任务 ➕ 2024-01-10 ⏳ 2024-01-15
- [ ] 使用 ⌛ 的任务 ➕ 2024-01-10 ⌛ 2024-01-15

## 格式要点总结

1. **任务必须以 `- ` 开头**，后跟复选框 `[ ]` 或其他状态符号
2. **优先级 emoji** 可以放在任务描述的任何位置
3. **日期 emoji** 后面必须跟 `yyyy-MM-dd` 格式的日期
4. **多个日期** 可以同时使用，如 `➕ 2024-01-10 📅 2024-01-15`
5. **标签** 使用 `#标签名` 格式
6. **链接** 支持 Obsidian 双向链接 `[[文件]]` 和 Markdown 链接 `[文本](url)`
