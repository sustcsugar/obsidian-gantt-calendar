## 💡 录制建议

### 准备工作
- 准备一些测试任务，覆盖各种状态、优先级、日期
- 确保有 Daily Note 文件和内容
- 准备一个有跨度的任务用于甘特图演示

### 录制技巧
- 鼠标移动要缓慢清晰
- 重要操作可以重复演示
- 使用屏幕放大功能突出细节
- 可以添加文字标注说明重点功能

### 时长控制
- 建议总时长控制在15-20分钟
- 每个功能演示时间不宜过长
- 可以准备精简版（5分钟）和完整版（20分钟）

### 任务格式示例

**Tasks 格式（emoji）**：
```markdown
- [ ] 🎯 任务标题 ⏫ ➕ 2025-01-10 📅 2025-01-15
```

**Dataview 格式（field）**：
```markdown
- [ ] 🎯 任务标题 [priority:: high] [created:: 2025-01-10] [due:: 2025-01-15]
```

**优先级**：
- 🔺 最高
- ⏫ 高
- 🔼 中
- ◽ 普通
- 🔽 低
- ⏬ 最低

**日期 emoji**：
- ➕ 创建日期
- 🛫 开始日期
- ⏳ 计划日期
- 📅 到期日期
- ✅ 完成日期
- ❌ 取消日期
